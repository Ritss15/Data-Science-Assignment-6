## Project Overview
This project demonstrates data visualization using Seaborn and Plotly.
It analyzes sales data to understand regional performance and relationships
between Quantity, Price, and Total Sales.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Plotly
- Google Colab

## Visualizations
1. Total Sales by Region (Bar Chart)
2. Sales Distribution by Region (Box Plot)
3. Correlation Heatmap

## How to Run
1. Open the notebook in Google Colab
2. Upload `sales_data.csv`
3. Run all cells
